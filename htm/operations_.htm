Operations Guide

Set Up Procedures

  1. Mount the front inner clamp on the skid using the supplied screws.
  2. Mount the rear inner clamp on the skid using the supplied screws.
  3. Place the manipulator in the clamps and install the outer clamps using the supplied screws. clamps.

  4. Remove the 9 Pin accessory port dummy plug from the accessory port.
  5. Install the 9 Pin connector from the Manipulator into the Accessory Port.

Do not overtighten the screws. The mount is designed to have a gap between the inner and outer

If you are using the manipulator with other accessories, it should be the last item plugged into the accessory port.

Use

The manipulator is operated using the hand controller.

The manipulator can be rotated out of the water so the jaws can open vertically, horizontally or at an arbitrary angle.

Depending upon the requirements, rotate the manipulator to be best suited gripping the intended object depending on whether the objects orientation is horizontal or vertical.

If the manipulator is in the fully vertical orientation it will be difficult to see how much it is open or closed. It is often best to offset the jaws at a slight angle so that you can see both jaws from the camera.
